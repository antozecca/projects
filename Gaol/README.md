# Gaol – Web Turn-Based Game

Gaol è un videogioco web fantasy turn-based in stile pixel-retrò, sviluppato come progetto universitario.

## Descrizione
Il gioco è basato su combattimenti a turni tra uno o più personaggi controllati dal giocatore e un nemico gestito dal sistema.  
Il gameplay è governato da un ciclo di turni discreto, con gestione dello stato di gioco, delle abilità e delle condizioni di vittoria o sconfitta.

## Caratteristiche principali
- Gameplay turn-based  
- Frontend multi-schermata (login, menu, lobby, gameplay, progressi)  
- Gestione dello stato dei personaggi e del ciclo dei turni  
- Persistenza dei progressi di gioco  
- Interazioni dinamiche e animazioni lato client  

## Tecnologie utilizzate
- HTML, CSS  
- JavaScript  
- PHP  

## Note
Il progetto si concentra sulla realizzazione di un’applicazione web interattiva, combinando logica di gioco, interfaccia grafica e gestione dei dati.
