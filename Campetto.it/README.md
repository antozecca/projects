# Campetto.it

Campetto.it è una piattaforma web per la ricerca e la prenotazione di impianti sportivi, sviluppata come progetto di tesi triennale in Ingegneria Informatica.

## Descrizione
L’applicazione consente agli utenti di individuare strutture sportive sul territorio, verificare la disponibilità dei campi e gestire le prenotazioni in modo semplice e intuitivo.  
Il progetto affronta problematiche reali tipiche dei servizi digitali, con particolare attenzione all’usabilità, alla gestione dei dati e all’organizzazione delle funzionalità.

## Funzionalità principali
- Gestione degli utenti  
- Ricerca di impianti sportivi  
- Prenotazione dei campi  
- Gestione dei dati e delle disponibilità  
- Interfaccia web orientata all’esperienza utente  

## Tecnologie utilizzate
- Ruby on Rails  
- HTML, CSS  
- PostgreSQL  

## Tesi
La cartella `Tesi/` contiene il materiale relativo alla tesi di laurea associata al progetto.
